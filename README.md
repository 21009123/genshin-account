# 🛠原神账号切换工具

一个用于在一台电脑上快速切换原神账号的小工具。

<del>原神周年可以领20连，为了方便切换各个小号领取，所以有了这个工具</del>

## 使用方法

环境要求：

* Windows 7 或更高版本。
* [.NET Framework 4.5](https://www.microsoft.com/zh-cn/download/details.aspx?id=30653) 或更高版本。

就三个按钮，先登录后保存，后续就可以随便切换了。

切换账户要关闭原神进程。
